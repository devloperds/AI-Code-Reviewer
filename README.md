AI Code Reviewer | MERN + Gemini 2.0 
automated code analysis platform that provides instant, context-aware feedback. 

Installation & Setup
Clone the Repository

Bash
git clone https://github.com/devloperds/ai-code-reviewer.git
cd ai-code-reviewer
Setup Backend

Create a .env file in the /backend directory.
Add your GEMINI_API_KEY.

Bash
cd backend
npm install
npm run dev
Setup Frontend

Bash
cd ../front-end
npm install
npm run dev
